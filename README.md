## 🥽 Meta 사이트 리뉴얼

이 페이지는 사용자에게 더 나은 사용 경험과 직관적인 소개 정보 전달을 목적으로 제작되었습니다.

로고의 컬러를 주조색으로 사용하여 **통일감**을 주었으며 섹션별로 영상을 첨부하여 **시선**을 **집중**시키도록 디자인되었습니다.
타이틀의 사이즈는 더욱 키워 **구분감**을 느끼게 하였고 중요키워드에는 다른 색상을 적용하고 fade in 효과를 넣어 **중요성**을 **강조**하였습니다.
